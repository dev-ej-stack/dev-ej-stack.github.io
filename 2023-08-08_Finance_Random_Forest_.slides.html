Building a Backtesting System

Construction d'un système de rétrotestage

In [30]:
def predict(train, test, predictors, model):
    # Fitting of the model
    model.fit(train[predictors], train["Target"])
    
    # predictions
    preds = model.predict(test[predictors])
    preds = pd.Series(preds, index=test.index, name="Predictions")
    combined = pd.concat([test["Target"], preds], axis=1)
    return combined

def backtest(data, model, predictors, start=2500, step=250):
    all_predictions = []
    # Predicting year by year, excluding the first 10 years.
    # Prédiction année par année, en excluant les 10 premières années.
    for i in range(start, data.shape[0], step):
        train = data.iloc[0:i].copy()
        test = data.iloc[i:(i+step)].copy()
        predictions = predict(train, test, predictors, model)
        all_predictions.append(predictions)   
    return pd.concat(all_predictions)

def predict(train, test, predictors, model):
    model.fit(train[predictors], train["Target"])
    # Enhancing control to establish how to generate 1 and 0 values 
    # based on the probability of the price going up.
    # Améliorer le contrôle pour établir comment générer les valeurs 1 et 0 
    # en fonction de la probabilité que le prix augmente.
    preds = model.predict_proba(test[predictors])[:,1]
    # Threshold - Impact to Reduce the Number of Trading Days
    # Indicates the days when making a trade is a genuine opportunity
    # Seuil - Impact pour réduire le nombre de jours de trading
    # Indique les jours où effectuer un échange est une véritable opportunité
    preds[preds >=.6] = 1
    preds[preds <.6] = 0
    preds = pd.Series(preds, index=test.index, name="Predictions")
    combined = pd.concat([test["Target"], preds], axis=1)
    return combined

Grid search is performed to find the best hyperparameters, and then the final model is trained with the best hyperparameters using the full training set. Predictions are made on the test set, and the model's performance is evaluated.

Une recherche par grille est effectuée pour trouver les meilleurs hyperparamètres, puis le modèle final est entraîné avec les meilleurs hyperparamètres en utilisant l'ensemble complet d'entraînement. Des prédictions sont effectuées sur l'ensemble de test, et la performance du modèle est évaluée.

A good practice is to use cross-validation (in this case, cv=5) to evaluate model performance and select the best hyperparameters. This helps avoid overfitting to the training data and ensures a more accurate evaluation of the model.

Une bonne pratique consiste à utiliser la validation croisée (dans ce cas, cv=5) pour évaluer la performance du modèle et sélectionner les meilleurs hyperparamètres. Cela permet d'éviter le surajustement aux données d'entraînement et garantit une évaluation plus précise du modèle.

Possible improvements to a new model:

Améliorations possibles pour un nouveau modèle :

  • The S&P500 operates during a limited period, with open overnight exchanges.

  • Use other market indices around the world to correlate with those exchanges to understand price movements in relation to ups or downs.

  • Incorporate economic news.

  • L'indice S&P500 opère pendant une période précise de la journée, mais il a des bourses ouvertes la nuit.

  • Utiliser d'autres indices boursiers dans le monde pour corréler avec ces bourses et comprendre les mouvements de prix en relation avec les hausses ou les baisses.

  • Intégrer les actualités économiques.

  • Consider macroeconomic conditions such as inflation, interest rates, etc.

  • Add key components of the S&P500 index (key stocks and sectors) to identify trends.

  • Use finer data resolution (hourly/minute/second) for better prediction accuracy.

  • Prendre en compte les conditions macroéconomiques telles que l'inflation, les taux d'intérêt, etc.

  • Ajouter les composants clés de l'indice S&P500, tels que les actions clés et les secteurs clés, pour identifier les tendances.

  • Augmenter la résolution des données en utilisant des données horaires/minutes/secondes pour améliorer la précision des prédictions.
